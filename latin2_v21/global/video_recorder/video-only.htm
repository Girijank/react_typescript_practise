American Sign Language — Webcam Give It a Try tool

The Give It a Try tool allows you to practice by temporarily recording videos in the browser using your computer's webcam. Videos are not stored on any servers, so if you want to keep one or two, make sure you save them.

Instructions

  • Select the camera icon, or press "a," to preview the camera.
  • Select "Start Recording," or press "r," to start recording immediately.
  • Select "Start Recording," or press "r," to start recording.
  • Select "Stop Recording," or press "r," to stop recording.
  • Use video controls such as "play" to view the video.
  • Select "Save," or press "s," to download the video.
  • Select "Start Recording," or press "r," to start recording again.
  • Press "a" to preview the camera.
Note: This browser does not support recording video. If available, use the built-in camera app on your device instead.
