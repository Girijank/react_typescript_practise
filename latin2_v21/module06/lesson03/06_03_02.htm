Contexto Conprensio!

Apollo’s Aim: oportet, oportere, oportuit, --

This verb only appears in the 3rd person singular. Here is how it looks:

Tense Latin Translation
Present oportet It is fitting; it is necessary
Imperfect oportebat It was fitting; it was necessary
Future oportebit It will be fitting; it will be necessary
Perfect oportuit It has been fitting; it has been necessary
Pluperfect oportuerat It had been fitting; it had been necessary
Future Perfect oportuerit It will have been fitting; it will have been necessary
