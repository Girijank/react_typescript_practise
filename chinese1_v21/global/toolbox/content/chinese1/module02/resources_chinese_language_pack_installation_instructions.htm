Microsoft Windows Chinese Language Pack Installation Instructions

  • Update your browser to the latest version. If you are using Microsoft Edge, you can download the latest version at https://www.microsoft.com/en-us/edge .
  • Update Language Support Options:
    1. In Windows, click Start menu;
    2. Type “region” and choose “Region Setting” from the list;
    3. Click on "Language” tab;
    4. Click on "Add a language";
    5. Select “Chinese (Simplified, China)” and click “OK”;
    6. Click “Next”, click checkbox for “Install language pack” (Note: you can also click checkboxes “Text-to-speech” and “Handwriting” if you wish to explore these features on Chinese characters in the future), then click “Install”;
    7. Once the installation is complete, “Chinese (Simplified, China)” will appear on the list of your preferred languages.
