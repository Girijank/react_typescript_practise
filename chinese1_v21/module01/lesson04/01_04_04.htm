打汉字 Dǎ hànzì Typing Chinese Characters

With modern technology, people are typing more often than writing by hand. That’s why you also need to understand how to type Chinese characters on computers or smartphones.

First you will install the Chinese input method on the computer. Below are the installation instructions for different operating systems.

Review the Instructions on how to install Chinese input method on your device.

Once installed, let’s see how to use it.

1

Switch input method from English to Chinese Simplified Pinyin.

2

In an open document, type the pinyin letters.

3

Among the characters displayed, find the character that you need, type the number before it, then the character appears.
