Now you are almost there to be able to discuss destinations and how to get there, using complete sentences in Chinese. Before you can put things in action, let’s learn some grammar points.


Using 怎么 zěnme (how) to ask for directions

怎么 zěnme (how), when used in a sentence, should be placed right before the verb. You will need this for your writing assignment later this lesson.

For example:

  • 你们怎么去图书馆？Nǐmen zěnme qù túshūguǎn?
    How do you go to the library?
  • 去博物馆怎么走？Qù bówùguǎn zěnme zǒu?
    How do I get to the museum?
Saying address in Chinese

When saying an address in Chinese, remember the rule of going from big to small, which is the opposite of saying an address in English.

city + 市 shì + road + 路 lù + house number + 号 hào

Note :

  • You can add more information such as country, province, and district if needed, all in order from big to small.
  • You will learn more address-related words in the future, such as “province,” “drive,” “boulevard,” and “zip code.”
  • For the names of cities or roads in English, you can simply write them in English.

For example:

Orlando 市 Bluestar 路三十八号 Orlando shì Bluestar lù sānshíbā hào (#38 Bluestar Road, Orlando City)

Note: For the house number, you can also write it in numeral form instead of Chinese characters.

Describing destination and transportation in a sentence

Let’s write complete sentences to describe how to get somewhere by following the structure below. You will need this for your writing assignment later in this lesson.

Subject + 坐 zuò/骑 qí + transportation + 去 qù (to go to) + destination

For example:

  • 我们坐出租车去医院。Wǒmen zuò chūzūchē qù yīyuàn. (We ride the taxi to the hospital.)
  • 她姐姐骑单车去学校。Tā jǐjie qí dānchē qù xuéxiào. (Her older sister rides the bike to the school.)
