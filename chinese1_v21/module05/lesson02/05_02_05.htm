房间和家务 Fángjiān hé jiāwù Rooms and Chores

Now you have the vocabulary on some furniture items used in bedrooms. It’s time to learn how to use them to describe your room in some detail and to talk about what bedroom chores you can do.


Using 有 yǒu with Furniture

You’ve learned how to use 有 yǒu (to have) to express possession and to introduce family members. You’ve also learned how to use it to introduce the structure of a house. Now you can use it to describe the furniture items in your bedroom.

For example:

我的卧室有床。Wǒde wòshì yǒu chuáng.
My bedroom has a bed.

Using 要 yào with Chores

You’ve learned how to use 要 yào (to want, to be going to) to order food or drinks and to discuss activities at school. You can also use it to describe chores you plan to do。

For example:

我今天要整理床。Wǒ jīntiān yào zhěnglǐ chuáng.
I am going to tidy up the bed today.
